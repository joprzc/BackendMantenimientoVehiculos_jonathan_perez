Listo para escanear.
Bluetooth (recomendado)
  1. Enciende el vehículo y el OBDLink.
  2. Empareja el dispositivo en el SO (Windows/Mac/Linux).
  3. Confirma el puerto serie detectado (ej: COM5 /dev/ttyUSB0).
`export OBD_PORT="/dev/ttyUSB0"`
`export OBD_VEHICLE_CODE="{{ vehiculo.placa }}"`
`python manage.py collect_obd --vehicle {{ vehiculo.id }}`
Wi‑Fi / TCP
  1. Conéctate a la red del OBDLink.
  2. Identifica IP y puerto (por defecto 192.168.x.x:35000).
  3. Prueba conectividad: `nc -vz $OBD_WIFI_HOST $OBD_WIFI_PORT`.
`export OBD_WIFI_HOST="192.168.0.10"`
`export OBD_WIFI_PORT="35000"`
`export OBD_VEHICLE_CODE="{{ vehiculo.placa }}"`
`python manage.py collect_obd --vehicle {{ vehiculo.id }}`
